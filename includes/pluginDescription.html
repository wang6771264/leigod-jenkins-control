Integrates Jenkins in Jetbrains IDEs like IntelliJ or CLion

Features
  • View Jobs
  • Trigger Jobs
  • Show Log for Job (including running)
  • Show Job Result as JUnit View
  • Multibranch support
  • Notifications: Settings -> Appearance -> Notifications -> Jenkins Notifications
  • Support of Leigod jenkins build and deploy
  • Build with history parameter
  • Git Push changes to build
